# Django: создание функциональных веб-приложений

Данный репозиторий содержит материалы и практические работы, выполненные в рамках курса **«Django: создание функциональных веб-приложений»**.

В ходе курса был изучен фреймворк Django и получены практические навыки разработки полноценных веб-приложений на Python, включая реализацию серверной логики, работу с базами данных и базовую интеграцию пользовательского интерфейса.
